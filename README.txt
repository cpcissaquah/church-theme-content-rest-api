Extends the Church Theme Content plugin to enable REST API functionality.

== Description ==

Enables REST API functionality for the [Church Theme Content](https://wordpress.org/plugins/church-theme-content/) plugin.  There is really no reason you would need to install or use t    his plugin if you are not familiar with REST.  Enabling the REST API will allow developers to write scripts that interact with Sermons and their metadata.

= Thanks =

Big thanks to Steven Gliebe with [Church Themes](http://churchthemes.com/) for providing the information to make this possible.

== Installation ==

Search for the *Church Theme Content - REST API* plugin using the *Add Plugin* tool from wordpress.

The Church Themes team has put together a very concise guide to installing plugins here:  [Plugin Installation](https://churchthemes.com/guides/user/plugins/plugin-installation/).  I h    ighly recommend it.

== Frequently Asked Questions ==

= What does this plugin actually do? =

It enables developers to use the [WP REST API](http://v2.wp-api.org/) with the [Church Theme Content](https://wordpress.org/plugins/church-theme-content/).
