Church Theme Content - REST API (WordPress Plugin Extension)
===========================================================
This WordPress plugin extends the [Church Theme Content](https://github.com/churchthemes/church-theme-content) (provided by [churchthemes.com](https://churchthemes.com)) to enable the use of the [WP REST API](http://v2.wp-api.org/).
